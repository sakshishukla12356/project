🌩️ AI-Powered Multi-Cloud Management System

🚀 Overview

A smart cloud management platform that integrates AWS and Azure, allowing users to monitor resources, track carbon emissions, and control services using an AI chatbot.



🔥 Features

* 🤖 AI Chatbot for cloud insights
* ☁️ Multi-cloud support (AWS + Azure)
* ⚡ One-click resource control (Stop services)
* 🌱 Carbon emission tracking
* 👤 Multi-user system with secure credential handling
* 📊 Dashboard for usage and analytics

---

## 🛠️ Tech Stack

* FastAPI (Backend)
* AWS SDK (boto3)
* Azure SDK
* SQLite / SQLAlchemy
* React (Frontend - in progress)



 🧠 How It Works

1. User connects cloud accounts
2. Credentials stored securely in DB
3. Chatbot fetches real-time data
4. Users can control services via UI



🔐 Security

* No hardcoded credentials
* User-based cloud access
* Environment variables protected


 🚀 Future Enhancements

* Auto resource optimization
* Cost prediction AI
* Start/Restart services
* Advanced analytics
